# CareRoot 🏥

> AI-powered health triage & hospital navigation website

## What it does
CareRoot lets users describe their symptoms, share their location, and instantly get:
- **AI severity assessment** (Low / Moderate / High / Emergency)
- **Personalized action steps** based on symptoms
- **Nearby hospitals** pulled live from OpenStreetMap
- **Interactive map** showing hospitals relative to the user
- **Get Directions** via Google Maps
